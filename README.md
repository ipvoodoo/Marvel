# Marvel app #

Приложение для просмотра характеристик и истории героев по комиксам Вселенной Marvel

### Примерный список используемых технологий: ###

* Navigation Component
* Retrofit
* OkHttp
* Dependency injection with Hilt
* Coroutines
* Kotlin Serialization
* DataStore
* Room
* Lifecycle
* Shimmer
* Clean Architecture
